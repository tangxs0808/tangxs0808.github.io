树莓派4b上手初体验，超频2.1GHz！

作为一名PC DIY玩家，对于Single Board Computer多多少少都会些兴趣，而作为SBC最广为人知的代表，Raspberry Pi更是我个人从公布之初就开始关注，我也在第二代的时候入手过一块B版，但因为是纯个人兴趣爱好，而非工作或者开发需要，自然到手把玩一番后就吃灰了，但其性能很弱，其实也是我对其没有继续折腾的一个重要原因。直到去年Raspberry Pi 4的推出，才完全改变了这块SBC孱弱性能的缺点。

Raspberry Pi 4搭载了博通BCM2711 SoC，在CPU部分采用了四核心的Cortex-A72，虽然相比如今主流的手机SoC，这配置其实也不够看，但对于一块SBC的性能，相信大多数玩家和用户都能感到满意了，我当时也着实心动了，但鉴于太多东西玩了，抽身乏术，便慢慢淡忘了。但在今年五月底，Raspberry Pi 4突然公布了个8GB内存版，就再次激起了我的兴趣，所以到最近有些时间，入手了一块来玩玩。

当然我也并非是“滚水禄脚”那样就下单了，在入手前我也是做了些功课的，首先是电源的购置。Raspberry Pi 4在去年刚推出不久，便曝出了USB-C口有设计缺陷，会与PD充电头不兼容，但后续出货的批次据说就改进了，所以这次8GB版理论上是不会有这个问题，而且这次新版还修改了电源模块（但没说是为了干嘛），官方表示为此也是延误了一些推出的时间，原本我还想着本来用手头有的PD充电头，但最终还是选择了一个号称专为Raspberry Pi做的5V/3A输出充电头，配合一条带有开关的USB-C充电线。

再就是散热，博通BCM2711 SoC用的是28nm工艺制程，CPU都是俗称的大核，虽然最高主频被限制在1.5GHz，但据一些已入手的用户反映，Raspberry Pi 4的发热还是挺大的，加上我比较无聊，想着买来要玩下超频…所以就特别加装一个带有双风扇主动散热的配件，网上还有用迷你塔式甚至水冷散热，但我觉得那些有点夸张了。而有意思的是，酷冷至尊在前段时间还推出了一个Raspberry Pi 4的机壳，宣称能提供压制超频后的发热，但那个壳目前只有在国外卖，看看后续能否借一个来体验下。

由于发展到第4代了，Raspberry Pi现在的操作系统、软件和教程等等什么的都非常成熟了，所以即使你不是开发者或创客（Marker），其实安装好官方的Raspberry Pi OS，把它作为一部超小型电脑，应付上上网、看看视频、打打文档等等使用，也是完全没问题的。

而且整个操作系统的安装方式也非常简单，从Raspberry Pi官网下载最新的固件，用同样是官方的Imager烧录进MicroSD卡里面，再把卡插进Raspberry Pi，插电开机就进入系统，没有比这个更容易的了，估计连我那位对电子设备一窍不通的女朋友，都应该知道怎样弄。

当然我还是想试一下比较进阶的玩法，主要是因为我觉得MicroSD卡槽的读写速度太慢了，所以想着把系统装到移动U盘或者SSD上，可以有更好的系统运行性能，但目前Raspberry Pi 4还没有原生的从USB启动支持（有个beta），好在有位高玩做了个BerryBoot，可以实现这种启动方式。

只是理论操作都很容易懂，但实际安装起来，我目前遇到了个BerryBoot不支持官方OS固件的问题，会提示不支持SquashFS，解决办法是转换下原版的img文件，但可能我们公司的网络有些问题，在Raspberry Pi OS里面无法安装kpartx相关文件，还无法转成能用的固件，故还要折腾下。

再就是超频了，据其他玩家的试验，在供电和散热足够情况下，Raspberry Pi 4这块SoC的CPU，超频到全核2.0GHz是很容易的，而有点意外的是，我发现PC DIY玩家喜欢的国外媒体tomshardware，居然也出过一篇Raspberry Pi超频教程，但现在超频还有个更简单的方式，有玩家做了图形界面的CammanderPi，在上面点几点就能超频了。只是在安装这个软件时，我同样遇到上面的连不上Raspberry服务器的情况，所以也暂时没超到频（主要是想先写这篇文章）。

已超频至2.1GHz的CPU，750MHz的GPU，稳定运行。
